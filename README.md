# SaaS Onboarding UI

A product-focused onboarding interface for B2B SaaS teams that need to turn a new customer kickoff into a clear launch plan. The experience centers on guided setup, progress visibility, and team handoffs so customers can move from sales close to go-live with fewer loose ends.

Built with Next.js, TypeScript, and Tailwind CSS, this project is designed as a polished portfolio piece that feels closer to a real SaaS product surface than a starter template.

## What This Demonstrates

- Product-minded UI copy and page structure for a SaaS onboarding workflow
- Responsive layout decisions for landing, feature, and conversion sections
- Reusable component patterns for buttons, cards, navigation, and page sections
- TypeScript-first organization with shared types and utility modules
- Maintainable styling through Tailwind CSS and project-level design tokens

## Features

- Conversion-focused hero section with clear customer onboarding positioning
- Onboarding progress preview that communicates task completion at a glance
- Feature cards for guided setup, progress signals, and team handoffs
- Focused call-to-action section for starting a customer journey
- Shared UI primitives for consistent buttons and cards
- App Router structure ready for additional product screens

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Check types and linting:

```bash
npm run typecheck
npm run lint
```

Build for production:

```bash
npm run build
```
